# Freelance Automation — AI Agent for Freelance Marketplaces

Исследование: возможно ли использовать Hermes для автоматического заработка на фриланс-биржах (fl.ru, kwork, freelance.habr, work-zilla, upwork).

## Ключевой вывод

**Полностью автономная система — невозможна.** CAPTCHA, антибот-системы и юридические риски (биржи запрещают автоматизацию) делают полную автоматизацию нереализуемой. Как **полуавтомат под контролем человека** — Hermes может взять на себя 70-80% рутины.

## Что Hermes может делать

| Функция | Реализация | Сложность |
|---------|-----------|-----------|
| Мониторинг новых заказов | browser_navigate + browser_snapshot по расписанию (cron) | 🟡 |
| Анализ ТЗ | LLM → анализ требований | 🟢 |
| Генерация отклика | LLM → текст, адаптированный под задачу | 🟢 |
| Ведение диалога с клиентом | Telegram gateway (если биржа → браузер) | 🟡 |
| Выполнение задач | Инструменты Hermes (код, тексты, файлы, email) | 🟢 |
| Отправка результата | browser / email | 🟡 |

## Чего Hermes не может (нужен человек)

| Задача | Причина |
|--------|---------|
| Регистрация и верификация аккаунта | SMS/2FA, паспортные данные |
| Прохождение CAPTCHA | Требуется внешний сервис (2captcha, антикапча) |
| Физический договор/акт | Юридическое лицо, ЭЦП |
| Получение денег на карту | Банковская карта, самозанятость/ИП |
| Обход антибот-систем fl.ru/kwork | Анализ поведения, тайминги, прокси |
| Восстановление рейтинга после ошибки | Человеческий фактор |

| Биржа | Публичный API | Возможность автоматизации |
|-------|:------------:|---------------------------|
| fl.ru | ❌ Нет | Только через браузер |
| kwork.ru | ❌ Нет | Только через браузер |
| freelance.habr | ❌ Нет | Только через браузер |
| work-zilla.com | ❌ Нет | Только через браузер |

Все биржи активно борются с автоматизацией: отслеживают скорость действий, паттерны кликов, тайминги между действиями.

## Архитектура (полуавтомат)

```
Ты (контроль) ←→ Telegram ←→ Hermes Agent
                                 ↕
                            Browser Automation
                                 ↕
                         Фриланс-биржа (fl.ru / kwork)
```

**Жизненный цикл заказа:**
1. **Мониторинг:** cron job раз в N минут проверяет новые заказы по ключевым словам
2. **Отбор:** Hermes анализирует ТЗ, отсеивает неподходящие
3. **Отклик:** показывает тебе → ты одобряешь → Hermes отправляет
4. **Диалог:** Hermes ведёт переписку, ты контролируешь
5. **Выполнение:** Hermes делает работу (код/текст/анализ)
6. **Сдача:** Hermes отправляет результат
7. **Контроль:** ты проверяешь и закрываешь заказ

## Потенциальный заработок (Россия, 2025-2026)

### Типы задач, подходящих для автоматизации

| Тип задачи | Средний чек | Время Hermes | В день |
|------------|:-----------:|:------------:|:------:|
| Тексты / статьи / рерайт | 500-2 000 ₽ | 10-20 мин | 5-10 |
| Парсинг данных (Python) | 1 000-5 000 ₽ | 20-40 мин | 3-6 |
| Настройка ботов / автоматизация | 3 000-15 000 ₽ | 1-3 часа | 1-2 |
| Excel / Google Sheets | 300-1 000 ₽ | 5-15 мин | 8-15 |
| Техподдержка / модерация | 200-500 ₽/час | постоянно | в чате |
| Переводы | 300-800 ₽ | 10-20 мин | 5-8 |

### Сценарии

**Осторожный** (3-5 заказов/день, средний чек 500-800 ₽):
- Дневная выручка: 1 500 — 4 000 ₽
- Месячная (22 дня): 33 000 — 88 000 ₽
- Минус комиссия биржи (10-20%) + антикапча + прокси
- **Чистыми: ~25 000 — 70 000 ₽/мес**

**Умеренный** (5-8 заказов/день, средний чек 1 000-3 000 ₽):
- Дневная выручка: 5 000 — 24 000 ₽
- Месячная: 110 000 — 528 000 ₽
- **Чистыми: ~80 000 — 400 000 ₽/мес**

### Риски
- Блокировка аккаунта за автоматизацию (потеря рейтинга и отзывов)
- Необходимость нескольких аккаунтов для масштабирования
- Правовые риски: использование ботов запрещено правилами всех российских бирж
- Репутационные риски: клиент заметит роботизированный стиль

## Что делать для запуска

1. Создать аккаунт на бирже (ты)
2. Настроить Hermes для доступа к бирже через browser
3. Написать cron-job для мониторинга заказов
4. Настроить шаблоны откликов под разные категории
5. Подключить сервис антикапчи (2captcha, capsolver)
6. Настроить прокси (резидентные, мобильные)
