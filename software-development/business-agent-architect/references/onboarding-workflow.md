# Перед началом работы с клиентом

1. **Спросить Олега:**
   - Тип бизнеса (салон/магазин/услуги)
   - Какие каналы нужны (Telegram/сайт/соцсети)
   - Какая рутина бесит больше всего (запись/модерация/CRM/FAQ)
   - Бюджет
   - Используют ли уже CRM? Какую?

2. **Выбрать шаблон:**
   - Нужны только ответы на вопросы → faq-bot
   - Нужна модерация соцсетей → moderation-bot
   - Нужна запись + CRM → crm-booking-bot

3. **Скопировать шаблон:**
   ```
   cp -r ~/agents/templates/{template-name}/ ~/agents/{client-name}/
   ```
4. **Настроить .env** (токен, данные компании, БД)

5. **Проверить по чеклисту** (references/checklist.md)
