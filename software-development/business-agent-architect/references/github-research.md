# Результаты исследования GitHub (2026-05)

## Лучшие шаблоны Telegram ботов

### 1. AiogramBotTemplate (⭐128)
**Стек:** aiogram + aiogram-dialog + Dishka DI + Pydantic + FastAPI + SQLAlchemy + asyncpg + Alembic + Docker + PostgreSQL + Redis + UV + Ruff
**Ссылка:** https://github.com/arturboyun/AiogramBotTemplate
**Когда брать:** Для production-агента с высокими требованиями к качеству кода, DI, тестами

### 2. TGB-Booking / BookingServices (⭐2)
**Стек:** aiogram 3 + async SQLAlchemy + SQLite + Alembic
**Ссылка:** https://github.com/WingorOsnova/TGB-Booking
**Когда брать:** Для быстрого старта бота записи в салон/барбершоп. Русский язык. Есть админ-панель в Telegram. Берём как базу, переписываем под себя.

### 3. fastapi-aiogram-template (⭐1)
**Стек:** FastAPI + aiogram 3 (webhooks)
**Ссылка:** https://github.com/hexfrost/fastapi-aiogram-template
**Когда брать:** Если нужен минимальный шаблон для вебхуков

### 4. aiogram-fastapi-bot-template (⭐5)
**Стек:** aiogram + FastAPI (webhooks)
**Ссылка:** https://github.com/BushlanovDev/aiogram-fastapi-bot-template
**Когда брать:** Если нужен простой старт с вебхуками

## AI/LLM Фреймворки для бизнес-агентов

### Проверенные инструменты:
- **Hermes Agent** — создание агентов (скиллы, промпты, память, cron)
- **OpenClaw** — оркестратор, gateway, маршрутизация между агентами
- **DeepSeek V4 Flash** — основная LLM (доступна, быстрая, дёшево)
- **Yandex GPT** — запасной вариант (российский, не заблокируют)

### Вспомогательные библиотеки:
- **python-telegram-bot** (⭐29k) — классика
- **aiogram** (⭐5.7k) — асинхронный, современный (рекомендую)
- **FastAPI** — для API и вебхуков
- **PostgreSQL** — основная БД

## Что не нашлось (ниши)
- Open-source AI-агентов для модерации соцсетей под малый бизнес
- Готовых решений «под ключ» для Крыма
- Шаблонов с DeepSeek интеграцией для Telegram
- State-machine фреймворков, заточенных под бизнес-агенты
